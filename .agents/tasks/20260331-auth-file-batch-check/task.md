# 任务说明

## 目标

在认证文件管理中新增批量检查能力，支持汇总展示可用认证文件、综合剩余额度、启用/禁用数量等核心信息，并通过弹窗查看详细检查结果。

## 范围

- 后端新增批量检查与汇总接口
- 前端认证文件页新增汇总区、批量检查入口、详情弹窗
- 开发实例切换到可验证自有前端改动的资源来源

## 非目标

- 不在本次引入新的前端测试框架
- 不重构现有所有单文件 quota 逻辑
- 不扩展到所有 provider，仅覆盖当前已有成熟 quota 逻辑的 provider

## 验收

- 能在认证文件页触发批量检查
- 页面能展示总数、启用、禁用、可用状态与综合剩余额度等核心信息
- 支持弹窗查看单文件检查详情
- 本地开发实例能验证到当前前端/后端改动
